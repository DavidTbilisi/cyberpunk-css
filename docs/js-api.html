JavaScript API

Init

The repo's cyberpunk.js exposes a global CyberUI object with an init() entrypoint used by the demo. Call it after DOM load.

<script src="cyberpunk.js"></script>
<script>
  window.addEventListener('load', () => CyberUI.init());
</script>

SFX

There is an internal WebAudio SFX engine. Use:

CyberUI.sfx.play('click');
CyberUI.sfx.play('keypress');
CyberUI.sfx.play('success');
CyberUI.sfx.play('error');
CyberUI.sfx.play('open');
CyberUI.sfx.play('close');

Note: browsers require a user gesture to unlock audio. The demo includes an audio unlock helper that resumes the AudioContext on first pointer/key interaction.

Helpers

Useful functions (examples in cyberpunk.js):

  • CyberUI.toast(message, variant) — shows a toast in the stack
  • CyberUI.openModal(id) / CyberUI.closeModal(id) — modal helpers

Examples

Show a toast programmatically:

CyberUI.toast('Mission complete', 'success');

Play a sound and open a modal:

CyberUI.sfx.play('open');
CyberUI.openModal('demo-modal');

Testing & Debug

Open the browser console and call the SFX or helpers directly to test without UI interactions.

CyberUI.sfx.play('click');
CyberUI.toast('Test', 'success');
