Styles & Variables

Color palette

SCSS variables are defined in _variables.scss. At runtime we expose CSS custom properties in _base.scss as --cp-green, --cp-cyan, --cp-magenta, --cp-purple, --cp-blue, --cp-amber.

GREEN
var(--cp-green)
CYAN
var(--cp-cyan)
MAGENTA
var(--cp-magenta)
PURPLE
var(--cp-purple)
BLUE
var(--cp-blue)
AMBER
var(--cp-amber)

Example usage in HTML:

<div style="background:var(--cp-magenta);padding:12px;border-radius:6px;">MAGENTA</div>

Mixins

Useful mixins:

  • @include retro-border($color) — neon border + inset glow
  • @include retro-glow-text($color) — text glow and shadow
  • @include grid-bg — subtle grid background

Example: panel with mixins

@include retro-border($cp-purple);
@include grid-bg;
background: rgba(0,0,0,0.5);
padding: 16px;

Build

Compile SCSS to CSS (Dart Sass recommended):

npx sass cyberpunk.scss cyberpunk.css --style=compressed
