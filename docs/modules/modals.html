Modals Module

File: scripts/modals.js

Purpose

Open and close modal dialogs with backdrop handling and keyboard support; supports confirm-style modals used for destructive actions.

Data attributes

  • data-modal-open="modal-id" — elements that open the modal with the matching id.
  • data-modal-close — elements that close the current modal.

API

  • openModal(id) — show modal by id (adds `is-open` class and traps focus).
  • closeModal(id) — hide modal and restore focus.
  • initModals() — attach attribute-based listeners (module auto-initializes on DOMContentLoaded).

Accessibility notes

  • Module attempts to manage focus by moving it into the modal when opened and restoring focus on close.
  • Escape key closes open modals.

Tests

See tests/modals.test.js for open/close behavior, backdrop click, and confirm modal flow.
