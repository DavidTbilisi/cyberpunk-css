NetGraph Module

File: scripts/netgraph.js

Purpose

Lightweight SVG-based renderer for simple network topologies. Nodes and edges are drawn as SVG groups; an optional packet emitter animates packets along edges for demo purposes.

API

  • renderNetGraph(container, topology, options) — render nodes and edges. topology is an object containing nodes: [{id,x,y,label}] and links: [{from,to}].
  • _createSvg() and other internals are exported for testability.

Topology format

{
  nodes: [ { id: 'a', x: 50,  y:100, label:'Server A' }, ... ],
  links: [ { from: 'a', to: 'b' }, ... ]
}

Options

  • packetInterval — how often demo packets are emitted (ms)
  • nodeRadius — visual radius for nodes

Tests

See tests/netgraph.test.js — tests cover SVG creation, node/edge counts, and packet emitter behavior.
