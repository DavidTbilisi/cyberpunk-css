HexViewer Module

File: scripts/hexviewer.js

What it does

Renders a byte buffer into a readable hex + ASCII table and supplies a "copy hex" helper for easy clipboard copying.

API

  • renderHex(element, buffer) — render the given ArrayBuffer / Uint8Array / string into the provided container element.
  • Helpers exported for testing: _toUint8Array() and _padHex().

Usage

import { renderHex } from './scripts/hexviewer.js';

const el = document.querySelector('#hex');
fetch('/some.bin')
  .then(r => r.arrayBuffer())
  .then(buf => renderHex(el, buf));

Clipboard note

The module uses navigator.clipboard.writeText() for the "copy hex" action; in tests and older browsers this is mocked or polyfilled. When serving over insecure origins, clipboard writes may be blocked.

Tests

See tests/hexviewer.test.js — tests include TextEncoder/TextDecoder polyfill usage and clipboard mocking.
