Toasts

Class: .cp-toast — used by scripts/toasts.js to display transient notifications.

Example markup

<button data-toast="Downloaded">Download</button>

// JS will append a .cp-toast element to the toast container

Styling notes

  • Toasts use enter/exit transitions; match durations in CSS and JS when tuning.
  • Consider different background colors for success/warn types (modifier classes or attribute-driven styling).
