Buttons

Class: .cp-btn — used across the UI for interactive buttons.

Visual examples

Markup

<button class="cp-btn">Primary</button>
<button class="cp-btn" data-click-text="Sent">Send</button>
<button class="cp-btn" data-press-scale="0.95">Press</button>

Behavior notes

  • Ripples are added by scripts/buttons.js on click.
  • data-click-text will temporarily change the button's text.
  • data-press-scale reduces transform scale during press for tactile feedback.

Accessibility

Keep button contents simple text for reliable announcement by screen readers. If using icons, include an accessible label via aria-label.
