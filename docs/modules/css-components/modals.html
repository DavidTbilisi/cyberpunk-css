Modals

Class: .cp-modal — toggled with .is-open.

Markup

<div id="confirm" class="cp-modal">
  <div class="dialog" role="dialog" aria-modal="true">
    <h2>Confirm</h2>
    <button data-modal-close>Cancel</button>
    <button data-modal-close>OK</button>
  </div>
</div>

Behavior notes

  • Backdrop click and Escape should close modals (handled in JS).
  • Focus should be trapped inside the modal when open; ensure meaningful focus targets inside the dialog.
