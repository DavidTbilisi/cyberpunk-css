NetGraph

Class: .netgraph — SVG container styled for the demo.

Usage

<div class="netgraph" id="net"></div>
  <script type="module">import { renderNetGraph } from '../../scripts/netgraph.js';</script>

Styling notes

  • SVG elements inside the netgraph inherit stroke/fill from CSS variables — adjust variables in _variables.scss.
  • Animations are GPU-friendly transforms when possible.
