Navbar Module

File: scripts/navbar.js

Purpose

Handles scroll-aware UI state for the top navigation (adds/removes is-scrolled class).

Behavior

  • Listens to scroll and toggles a class on the document's body (or configured nav element) when the page is scrolled beyond a small threshold.
  • Helps drive CSS effects like drop shadows, blur, or a color shift on the navbar when the page moves.

Customization

Default threshold and target element are internal constants. If you need custom behavior, import the module and call its exported init function and pass a config object (see test file for examples):

// pseudo
import { initNavbar } from './scripts/navbar.js';
initNavbar({ target: document.querySelector('.site-header'), threshold: 12 });

Tests

Check tests/navbar.test.js for scroll simulations and class toggling assertions.
