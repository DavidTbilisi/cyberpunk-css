Toasts Module

File: scripts/toasts.js

API

  • showToast(message, options) — create and display a transient toast. Options: duration (ms), type (info/warn/success).
  • createToastElement(message, options) — returns a DOM element without auto-show.

Usage

import { showToast } from './scripts/toasts.js';
showToast('Saved', { duration: 2000, type: 'success' });

Wiring

Elements with data-toast will trigger toasts; example:

<button data-toast="Download started">Download</button>

Implementation notes

  • Toasts are appended to a container element created by the module and removed after a timeout.
  • Module uses CSS transitions; tests mock timers where appropriate.

Tests

See tests/toasts.test.js — tests cover element creation, display timing, and removal.
