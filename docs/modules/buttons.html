Buttons Module

File: scripts/buttons.js — Interactive button behaviors

Purpose: Small helpers that wire interactive button UX: ripples, data-driven text changes, press-scale, and smooth scroll triggers.

Features / Data attributes

  • Ripple — click ripple effect added to `.cp-btn` elements.
  • data-click-text — replaces button text on click with the provided value.
  • data-hover-text & data-default-text — temporary text on hover.
  • data-press-scale — scale factor (e.g. `0.95`) applied on mousedown and removed on mouseup/mouseleave.
  • data-scroll-to — scrolls the window to a vertical pixel position (`data-scroll-to="500"`) smoothly.

Usage examples

<button class="cp-btn" data-click-text="SENT">Send</button>

<button class="cp-btn" data-hover-text="Ready" data-default-text="Hover Me">Hover Me</button>

<button class="cp-btn" data-press-scale="0.96">Press Me</button>

<button class="cp-btn" data-scroll-to="800">Scroll Down</button>

Initialization

The module auto-initializes on DOM ready and attaches event handlers to elements with the above data attributes. To manually initialize, import the module and call its init function (see tests for examples).

Implementation notes

  • Ripple elements are transient DOM nodes removed after animationend.
  • Text replacement uses the element's textContent; keep markup simple.
  • Scroll uses window.scrollTo({ top, behavior: 'smooth' }) — mocked in tests.

Tests

See tests/buttons.test.js for unit coverage and expected behaviors (ripple creation/removal, attribute handling, scroll call).
