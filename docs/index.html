RETROPUNK UI — Documentation

Welcome. This documentation folder contains quick reference pages for the RETROPUNK UI framework bundled in this repository.

Getting started

  1. Build the CSS (if you edit SCSS): 
    npx sass cyberpunk.scss cyberpunk.css --style=compressed
  2. Open these docs or the index.html demo in a browser. To serve locally: 
    python -m http.server 8000
# open http://localhost:8000/docs/
  3. Scripts: cyberpunk.js exposes CyberUI and SFX helpers (see JS API page).

Terminal Example

Use the .cp-terminal component to display console-like output in the docs. This example demonstrates header controls, prompt lines and muted output.

system@retropunk:~/project
~$ npx sass cyberpunk.scss cyberpunk.css --style=compressed
Compiling SCSS... 12 files
✔ cyberpunk.css generated
Run python -m http.server to serve the demo locally.
<div class="cp-terminal">
  <div class="cp-terminal__header">...</div>
  <div class="cp-terminal__body">
    <div class="cp-terminal__line"><span class="cp-terminal__prompt">~$</span> npx sass cyberpunk.scss cyberpunk.css --style=compressed</div>
  </div>
</div>
